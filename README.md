# Portfolio Frontend (React)

This is the frontend of my personal portfolio website built using React and Vite.  
It displays my profile, projects, skills, and contact information.

## 🚀 Tech Stack
- React.js
- Vite
- CSS
- Axios
- React Router

## 📂 Features
- Responsive portfolio UI
- Project showcase
- Profile information section
- API integration with backend
- Modern component-based structure

## 🔧 Installation

Clone the repository:

git clone https://github.com/your-username/portfolio-frontend-react.git

Go to project folder:

cd portfolio-frontend-react

Install dependencies:

npm install

Start development server:

npm run dev

The app will run on:

http://localhost:5173

## 🔗 Backend API
This frontend connects to the backend API:

http://localhost:5000
